# Branding and Naming Policy

The following identifiers are reserved by the project owner:

- `Track Description Annotator`
- project logos and icons
- official screenshots and release artwork
- the official GitHub release repository and release pages

## You May Not

- use the project name, logo, or icon to imply that a modified build is an official release
- publish repackaged or modified builds under the same official name without permission
- present your own distribution channel as the official project page

## Descriptive Use

You may reference the project name when truthfully describing compatibility, review, or commentary, as long as you do not imply endorsement or official affiliation.
