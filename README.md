Track Description Annotator is a cross-platform desktop tool for text annotation on multi-object tracking data.

This repository is the official binary distribution channel. It does not publish source code.

## 📦 Downloads

Download the latest official builds from:

- [Releases](https://github.com/sky-creater/Track-Description-Annotator-app/releases)

- Windows `.exe`
- macOS `.dmg`
- Linux `.AppImage`

Each release should also include `SHA256SUMS.txt` for checksum verification.

## ✅ Official Release Policy

Only release assets attached to this repository are official builds.

- Source code is maintained separately in a private repository.
- Modified or repackaged versions published elsewhere are not official unless explicitly stated by the copyright holder.
